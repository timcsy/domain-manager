API 文件

REST API 和 MCP 端點說明

認證方式

Session Token

登入後取得，透過 X-Session-Token header 傳遞

API Key

程式化存取，透過 X-API-Key header 傳遞

端點列表

GET /api/v1/domains 列出所有域名
POST /api/v1/domains 建立域名
GET /api/v1/domains/{id} 取得域名詳情
PUT /api/v1/domains/{id} 更新域名
DEL /api/v1/domains/{id} 刪除域名
GET /api/v1/certificates 列出所有憑證
GET /api/v1/certificates/expiring 即將到期的憑證
POST /api/v1/certificates/{id}/renew 續期憑證
GET /api/v1/services K8s 服務列表
GET /api/v1/settings 系統設定
GET /api/v1/api-keys API 金鑰列表
GET /api/v1/backup 備份列表
POST /api/v1/backup 建立備份
GET /health 健康檢查（公開）

使用範例

# 使用 API Key 列出域名

curl -H "X-API-Key: dm_your_key" http://localhost:8080/api/v1/domains

# 建立域名

curl -X POST http://localhost:8080/api/v1/domains \

-H "X-API-Key: dm_your_key" \

-H "Content-Type: application/json" \

-d '{"domain_name":"app.example.com","target_service":"my-app","target_port":8080}'

MCP 協議

MCP（Model Context Protocol）讓 AI 工具（如 Claude Desktop）能透過 JSON-RPC 2.0 操作域名管理器。

端點：POST /mcp

可用工具

list_domains 列出所有域名
create_domain 建立域名對應
list_services 列出 K8s 服務
list_expiring_certificates 即將到期的憑證
get_system_health 系統健康狀態

使用範例

# 初始化 MCP 連線

curl -X POST http://localhost:8080/mcp \

-H "Content-Type: application/json" \

-d '{"jsonrpc":"2.0","id":1,"method":"initialize",

"params":{"protocolVersion":"2024-11-05","capabilities":{}}}'
